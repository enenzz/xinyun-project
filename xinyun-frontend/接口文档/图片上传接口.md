# 图片上传接口文档

## 接口信息

| 项目     | 内容                      |
| -------- | ------------------------- |
| 接口名称 | 上传图片                  |
| 接口地址 | `/api/common/upload/image` |
| 请求方式 | POST                      |
| 请求格式 | multipart/form-data       |
| 接口说明 | 上传图片到云平台，返回访问 URL |

---

## 请求参数

| 参数名 | 类型 | 必填 | 说明                    |
| ------ | ---- | ---- | ----------------------- |
| file   | file | 是   | 图片文件                |
| type   | string | 否   | 图片类型：avatar(头像)、poster(海报)、other(其他)，默认 other |

---

## 响应格式

### 成功响应

```json
{
  "code": 200,
  "message": "上传成功",
  "data": {
    "url": "http://localhost:8080/temp/avatar/2024/03/24/0ebe16706ea44452b7031e1dbb62736f.jpg",
    "filename": "0ebe16706ea44452b7031e1dbb62736f.jpg",
    "size": 102400
  }
}
```

### 响应字段说明

| 字段名   | 类型   | 说明              |
| -------- | ------ | ----------------- |
| url      | string | 图片访问完整 URL（用于前端回显和注册时传递） |
| filename | string | 文件名            |
| size     | number | 文件大小（字节）   |

---

## 请求示例

```
POST /api/common/upload/image HTTP/1.1
Content-Type: multipart/form-data; boundary=----WebKitFormBoundary

------WebKitFormBoundary
Content-Disposition: form-data; name="file"; filename="avatar.jpg"
Content-Type: image/jpeg

[图片文件二进制数据]
------WebKitFormBoundary
Content-Disposition: form-data; name="type"

avatar
------WebKitFormBoundary--
```

---

## 错误响应

### 文件为空

```json
{
  "code": 400,
  "message": "上传文件不能为空",
  "data": null
}
```

### 文件类型错误

```json
{
  "code": 400,
  "message": "只支持图片文件",
  "data": null
}
```

### 文件大小超限

```json
{
  "code": 400,
  "message": "文件大小不能超过 2MB",
  "data": null
}
```

### 服务器错误

```json
{
  "code": 500,
  "message": "上传失败：网络错误",
  "data": null
}
```

---

## 使用说明

1. **前端回显**：使用响应中的 `url` 字段，直接绑定到 `<img>` 标签的 `src` 属性
2. **注册流程**：先调用此接口获取图片 URL，再将 URL 传递给注册接口
3. **文件限制**：
   - 文件类型：JPG、PNG、GIF
   - 文件大小：不超过 2MB
4. **图片处理**：后端会自动压缩图片并上传到云平台

---

## 注意事项

1. 临时文件默认保存 24 小时，超时会被自动清理
2. 注册成功后，临时文件会立即删除
3. 建议使用预上传方案，避免资源浪费
4. 生产环境必须使用 HTTPS
